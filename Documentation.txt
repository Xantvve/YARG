== Документация по игре YARG (Yet Another Roguelike-Game)

Действительна для
0.0.0

Легенда версии:
Фундаментальный релиз - Мажорная версия - Минорная версия

-=-=-= Устройство игры:
Игра работает через игровой цикл. Поочередно отрабатывают следующие части цикла:

> Обработчик ввода вызывает игровые функции.
> Игровые функции пересчитывают обстановку, и порождают события.
> Технические функции обрабатывают события.

-=-=-= Подключения:
К главному файлу пристыковывается единый файл с подключениями. Он выглядит так:

 - Подключения внешних библиотек
 - Модуль с мелкими определениями
 - Словарь функций (определения мелких декларативных функций)
 - Определение игрового экрана        -> реализация игрового экрана					 !!
 - Определение класса персонажа       -> реализация методов персонажа                !! сделать
 - Определение класса игровой системы -> реализация                                  !! реализации

-=-=-= Игровой экран
Задается классом GameScreen. Объект объявляется с 2 переменными (Width, height) размера окна.
В переменных также задан picture, 2-мерный массив char.
       Методы
redraw : изменяет picture, читая некоторое событие (GameEvent)                        !! сделать события
append : выводит picture непосредственно на экран
clear  : очищает экран.

-=-=-= Персонаж
Задается классом Player.
Переменные: level, health, attack, defense, experience. Еще x и y.

-=-=-= Игра
Игра выделяется в отдельный объект класса GameSystem. Объявляется адресом к файлу с игровым уровнем.
Имеет метод playGame, разворачивающий игровой уровень.
